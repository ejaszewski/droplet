# Droplet

Droplet is a program for computing mathematical constants using BBP-style formulas of the form:

$$
\alpha = \sum_{k=0}^{\infty}\left[\frac{1}{b^k} \sum_{j=1}^n \frac{a_j}{\left(nk+j\right)^s}\right]
$$
